## Printing Way 
Let's say **'Hello'** to Ruby. :D
```ruby
print "Hello World"
```
#### Output:
```Hello World```

Now I am try to Print my name with `Hello World`. üòé
```ruby
print "Hello World "
print "Sayanta Chy "
```
#### Output:
```Hello WorldSayanta Chy```

Lets's add new line a newline inside print method.
```ruby
print "Hello World\n"
print "Sayanta Chy"
```
#### Output:
```
Hello World
Sayanta Chy
```
Now let's try to add some complement besides my name üòè
But now here we use `puts " "`.
```ruby
print "Hello World\n"
puts "Sayanta Chy"
print "is cool"
```
#### Output:
```
Hello World
Sayanta Chy
is cool
```
#### Learning Notes:
Here notice that `print ` print a text without new line and `puts ` print a text with newline. That the difference betwwen `print` and `puts`.

Let's print a diagram using `puts` in console.
```ruby
puts "   /|"
puts "  / |"
puts " /__|"
```
#### Output:
```
  /|
 / |
/__|
```
